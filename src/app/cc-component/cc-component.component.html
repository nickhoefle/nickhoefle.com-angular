Collector's Corner

A movie, book, and board game conisseur's dream! Collector's Corner started as a capstone project for LaunchCode's LiftOff program. I have enjoyed continuing to refine, expand, and add features to this with my pal Rich.

Register, create your first book/movie/game collection, and you're off! Each of the three mediums have a search function to find favorites to add to personalized lists for archiving or showing to friends. Collections across all users are centralized on a view where users can see what others have created and find new favorites. Personalized SideNotes provide an opportunity to explain why a particular film belongs to a collection, or how much you love (or hate!) it.
